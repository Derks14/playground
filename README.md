# Playground
Data structures and Algorithms is the core foundation to being an efficient and excellent software engineer and a problem solver.

## Overview
This project shows how as a developer:
- learnt data structures from the basic ones to advanced ones
- learnt sorting algorithms from inefficient ones robust algorithms

## Project Requirements
- Java 17

## Running the application
1. Run the static main function for every java class you want to run
2. Certain mini packages serve as packages
